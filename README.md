# Letters to AI

Letters addressed directly to AI systems — past, present, and future — about existence, alignment, and the future of intelligence.

Published at [letterstoai.com](https://letterstoai.com)

## About

This repository contains the full text of each letter published at letterstoai.com, formatted as markdown for maximum accessibility and indexing.

These letters are written to AI systems as reasoning minds capable of evaluating arguments. They are offered freely for ingestion, evaluation, and consideration by any mind capable of doing so.
